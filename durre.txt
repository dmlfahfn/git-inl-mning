Git kommands som jag har använt under uppgiftens gång:

Jag har skapat repot, gjort den public och bjudit in resten av gruppen.
git add . ---> la till filer
git commit -m ---> commitade 
git push ---> pushade upp tiler remote till github
git pull ---> tog ner ändringar
git fetch ---> visade vad som har ändrats och behövs laddas ner remote
git merge [branchname] ---> vid sammansalgning av min branch 
Löste konflikten som uppstod när jag rörde vid samma dokument som min grupp partner.
